# EDUProfile Management System

A production-ready student profiling and academic record management system.

## 🚀 Quick Start
1. **Faculty Access**: Go to `/auth/login` to manage programs and students.
2. **Student Portal**: Go to `/student-portal/login` for evaluations and enrollment.
3. **Recovery**: Students can reset access keys using their ID number.

## 🛠 Features
- **Formal Evaluation sheets**: High-fidelity, printable academic prospectuses.
- **Student ID Recovery**: Instant password reset to Student ID.
- **Permanent Links**: Shareable public evaluation URLs.
- **Enrollment System**: Class registration and automated weekly timetables.
